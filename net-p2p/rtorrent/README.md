rTorrent
========

### Color Patch

[rtorrent.rc color configuration variables] Updated to use C++ std namespaces from tr1, requires >0.9.4

```
color_bg_active = 8 [background]
color_fg_active = 10 [foreground (text)]
color_bg_complete = 14
color_fg_complete = 8
color_bg_inactive = 8
color_fg_inactive = 9
color_bg_dead = 8
color_fg_dead = 11
```
